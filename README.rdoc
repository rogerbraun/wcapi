Vivien 'v0n' Didelot :)

== Usage

  client = WCAPI::Client.new :wskey => '[your_api_key]'
  infos = client.get_locations :type => "isbn", :id => "014330223X", :oclcsymbol => "ATNHU"

  infos.institutions.first[:copies] # => "1"
